シグナル判定ロジック（記事準拠・絶対水準ベース）

記事「市場予測指標で市場の動きを考えてみる」(株おじさん) のロジックそのまま： SKEW÷VIX (米国) / PCR÷VI (日本) の絶対水準でテールリスクと買い圧力を読む。 水準が高い = テールヘッジ全開 = 警戒、水準が低い = パニック終了 = 買い場。
ラベル水準意味
警戒> p90テールヘッジ買いが極端に厚い高値圏。下落リスク高い
弱含みp75 〜 p90警戒寄り、利食い検討
中立p25 〜 p75普通の状態
打診買いp10 〜 p25低水準、買い候補
買い場< p10パニック終了サイン、強・買いゾーン
※ 分位閾値は 3年(750日) のフルレンジで算出し全期間に適用。
※ 日経シグナル = PCR÷VI の絶対水準 (J-Quants 建玉から自前算出 + Yahoo 日経VI)
※ 米国シグナル = SKEW÷VIX の絶対水準 (Yahoo ^SKEW / ^VIX)
※ 旧バージョンの ROC10 + 状態マシン (買い→退避→売り→打) は廃止。

シグナル別 N225 フォワードリターン検証 (全データを後付けで集計)

指標の見方
・PCR (全体OI) = フロント限月の全プット建玉 ÷ 全コール建玉（メディア標準）。1.0以上で下落ヘッジ厚（弱気）、低位は強気・楽観。
 補助: PCR(OTM) = ATM±20%レンジ内のみ。壁分析・センチメントの細かい温度感に。
・日経VI = 投資家が想定する向こう30日の日経平均ボラ。20以下は穏やか、30超は警戒。
・PCR÷VI（スプレッド）= テールリスクと買い圧力を一体評価。記事の SKEW÷VIX に倣う日本版。
・モメンタム = スプレッドの ROC(10営業日) = (今日 ÷ 10営業日前 − 1) × 100 [補助表示]。
・シグナル = スプレッドの絶対水準を5段階に分類（記事準拠の素朴解釈）。
・市場指標 = USDJPY / VIX(米国) / TOPIX(1306) / 米10年金利。日経/オプション動向と合わせて環境判定に使用。
データ源: J-Quants V2（オプション建玉, T+1反映） / Yahoo Finance（^NKVI.OS, ^N225, 1306.T, USDJPY=X, ^VIX, ^TNX, ^GSPC, ^SKEW）。毎平日朝に自動更新。グラフ表示は2024-04-01以降。
