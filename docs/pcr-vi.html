シグナル状態遷移ルール（記事準拠の循環ステートマシン）

記事「市場予測指標で市場の動きを考えてみる」(株おじさん) の「プラスになれば買い／減速すれば売り」 を、4状態の単方向ステートマシンで実装。記事の用語(state)と、3年実証勝率に基づく表示ラベル(display)を併記。
遷移条件記事の意味表示ラベル
買い → 退避ROC10 が 2営業日連続で減少勢いに陰り中立 → 弱含み
退避 → 売りROC10 が 0以下 に転落「減速すれば売り」弱含み → 買い場
売り → 打ROC10 が 2営業日連続で上昇底打ち打診買い場 → 中立
打 → 買いROC10 が 0超 に復帰「プラスになれば買い」中立 → 中立
※ 戻り遷移（買い⇄退避 など）は禁止。必ず 買い→退避→売り→打→買い の順で循環。
※ 「2連続」「ゼロクロス」のしきい値は記事に明記なしのヒューリスティック。実データの勝率検証は下表参照。

実証ベースの解釈（なぜ「売り」が買い場なのか）

3年(2023-04〜2026-04, n=750営業日)で各状態の20日後 N225 リターンを集計したところ、記事の解釈通りに動いたのは 退避 のみで、 "売り"状態は記事とは逆の強力な逆張り買い場であることが判明（勝率 76.2%、ベース比 +10.9pp）。

メカニズム解説：
・"売り"状態 = ROC10(PCR÷VI) が 0以下 に転落し、さらに連続反発もまだない局面
・PCR÷VI が急速に低下する典型は VI が急騰するパターン（=ボラティリティ・スパイク=パニック）
・つまり投資家がプット建玉を慌てて取得／清算し、市場が短期的に売られすぎている時期
・歴史的に パニック直後の N225 は平均回帰するため、底値圏での逆張りロングが優位
・SKEW÷VIX の米国版でも同様の「VIX スパイク後 = 強・買いゾーン」パターンが観測される

記事との差：株おじさんの記事は 2025-03-24 の "退避→下落" 例を提示しているが、これは当方の計算でも完全に再現される（=計算式は記事と一致）。 ただし「売り」シグナル後の動きは記事の言う「売り」と逆の挙動が3年中の代表的パターンで、その理由が上記のパニック→反発メカニズム。

表示ラベルは「実際に勝率が高かった状態」をハイライトする方針。state=売り を 買い場 と表示し、 state=退避 を 弱含み と表示。state=買い／打 はほぼ baseline 同等のため 中立 表示にトーンダウン。

シグナル別 N225 フォワードリターン検証 (全データを後付けで集計)

指標の見方
・PCR (全体OI) = フロント限月の全プット建玉 ÷ 全コール建玉（メディア標準）。1.0以上で下落ヘッジ厚（弱気）、低位は強気・楽観。
 補助: PCR(OTM) = ATM±20%レンジ内のみ。壁分析・センチメントの細かい温度感に。
・日経VI = 投資家が想定する向こう30日の日経平均ボラ。20以下は穏やか、30超は警戒。
・PCR÷VI（スプレッド）= テールリスクと買い圧力を一体評価。記事の SKEW÷VIX に倣う日本版。
・モメンタム = スプレッドの ROC(10営業日) = (今日 ÷ 10営業日前 − 1) × 100。
 記事「市場予測指標で市場の動きを考えてみる」(株おじさん) のシグナル: 「プラスになれば買い、減速すれば売り」。
・市場指標 = USDJPY / VIX(米国) / TOPIX(1306) / 米10年金利。日経/オプション動向と合わせて環境判定に使用。
