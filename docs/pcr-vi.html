🧪 強弱5段階シグナル【試験】 — 2指標 AND 条件で edge を強化

なぜ SKEW と PCR の単独使用ではダメか
単一指標の極端値（p10/p90）だけ見ると、edge の大半は VIX/VI（=市場ボラ）の panic 検出から来る。 SKEW（機関のヘッジコスト）や PCR（投資家のポジション）が持つ 独自情報は捨てられる。 特に「市場は穏やかなのに機関だけがコッソリ暴落保険を買っている」という暴落予兆が、VIX が動いていないため検出できない。

3指標が答える質問はそれぞれ違う

指標何を測るか先行性
VIX / VI市場全体の予想ボラの大きさ同期（=リアルタイム panic 計）
SKEW下方テールの保険料（機関のヘッジ需要）+20〜60営業日（先行）
PCR投資家のポジション（建玉残高）同期〜やや遅行

AND 条件で組み合わせると何が起きるか

  • SKEW 極高 × VIX 極低（米国ストロングセル）= 「市場は穏やかだが機関が密かに警戒」 → calm before storm。SKEW の先行性が活きる場面。
  • VIX 極高 × SKEW 既低（米国ストロングバイ）= 「panic 真只中、機関のヘッジは既にアンワインド済み」 → 底値圏。VIX 単独では「panic だが底かどうか不明」だが、SKEW が低くなっていることで底打ちを確認。
  • PCR 極高 × VI 極低（日経ストロングセル）= 「市場穏やかなのに投資家のプット建玉が極端」 → 機関の SKEW 相当の警戒蓄積（日本版）。
  • VI 極高 × PCR 高（日経ストロングバイ）= 「ボラ急騰 + retail 含む全員パニック買い」 → capitulation 完了の確度高い。

既存の単独 p10/p90 シグナル（PCR÷VI / SKEW÷VIX 絶対水準）は買い側のみ機能していた（売り側も多少効くが弱い）。 今回の AND 条件で売り側に強烈な edge が出現し、SKEW と PCR を分子に持つ価値が実証された。

シグナル判定ロジック（記事準拠・絶対水準ベース）

記事「市場予測指標で市場の動きを考えてみる」(株おじさん) のロジックそのまま： SKEW÷VIX (米国) / PCR÷VI (日本) の絶対水準でテールリスクと買い圧力を読む。 水準が高い = テールヘッジ全開 = 警戒、水準が低い = パニック終了 = 買い場。
ラベル水準意味
売り> p90テールヘッジ買いが極端に厚い高値圏。下落リスク高い
退避p75 〜 p90警戒寄り、利食い検討
中立p25 〜 p75普通の状態
打診買いp10 〜 p25低水準、買い候補
買い< p10パニック終了サイン、強・買いゾーン
※ 分位閾値は 3年(750日) のフルレンジで算出し全期間に適用。
※ 日経シグナル = PCR÷VI の絶対水準 (J-Quants 建玉から自前算出 + Yahoo 日経VI)
※ 米国シグナル = SKEW÷VIX の絶対水準 (Yahoo ^SKEW / ^VIX)
※ 旧バージョンの ROC10 + 状態マシン (買い→退避→売り→打) は廃止。

シグナル別 N225 フォワードリターン検証 (全データを後付けで集計)

指標の見方
・PCR (全体OI) = フロント限月の全プット建玉 ÷ 全コール建玉（メディア標準）。1.0以上で下落ヘッジ厚（弱気）、低位は強気・楽観。
 補助: PCR(OTM) = ATM±20%レンジ内のみ。壁分析・センチメントの細かい温度感に。
・日経VI = 投資家が想定する向こう30日の日経平均ボラ。20以下は穏やか、30超は警戒。
・PCR÷VI（スプレッド）= テールリスクと買い圧力を一体評価。記事の SKEW÷VIX に倣う日本版。
・モメンタム = スプレッドの ROC(10営業日) = (今日 ÷ 10営業日前 − 1) × 100 [補助表示]。
・シグナル = スプレッドの絶対水準を5段階に分類（記事準拠の素朴解釈）。
・市場指標 = USDJPY / VIX(米国) / TOPIX(1306) / 米10年金利。日経/オプション動向と合わせて環境判定に使用。
データ源: J-Quants V2（オプション建玉, T+1反映） / Yahoo Finance（^NKVI.OS, ^N225, 1306.T, USDJPY=X, ^VIX, ^TNX, ^GSPC, ^SKEW）。毎平日朝に自動更新。グラフ表示は2024-04-01以降。
